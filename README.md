# Art by RilyRobo

This repository contains the personal portfolio website for RilyRobo, featuring artwork, comics, videos, games, commissions, and social links in a single responsive web experience.

## Overview

The site is built as a static website using plain HTML, CSS, and JavaScript. It includes:

- A home page with featured content
- A gallery of artwork
- Comics and video pages
- Game showcases
- Commission pricing and contact information
- Shop and social media links
- An about page with artist background

## Project Structure

- `index.html` — Main page structure and content
- `aboutme.html` — About page content
- `style.css` — Site styling
- `scripts/` — JavaScript for navigation, gallery, comics, commissions, games, and videos
- `images/` — Site assets and media
- `data/` — JSON data used by the site
- `comics/` — Comic CSV data

## Getting Started

Since this is a static website, you can view it locally by opening `index.html` in a browser.

If you prefer to use a local server, you can run one from the project folder, for example:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Notes

- No build step is required.
- The site uses dynamic content loaded from local JavaScript and data files.
- Some sections may require internet access for embedded media or external links.
